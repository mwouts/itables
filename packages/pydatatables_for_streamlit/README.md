This is the PyDataTablesRenderers frontend for [Streamlit](https://streamlit.io/)

It is build automatically when the PyDataTablesRenderers Python package is build,
and the result of the build is moved to `src/pydatatables/pydatatables_for_streamlit`,
cf. the `build` script.

This package does not need to be published on `npm`.
