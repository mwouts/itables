# References

PyAgGrid is a wrapper for the [AG Grid](https://www.ag-grid.com/) Javascript library, for Python. AG Grid Community is developed by [AG Grid Ltd](https://www.ag-grid.com/about/) and made available under an MIT license. It has an extensive [documentation](https://www.ag-grid.com/javascript-data-grid/getting-started/), as well as a large set of [examples](https://www.ag-grid.com/example/).

PyAgGrid is developed in the [ITables](https://github.com/mwouts/itables) project, together with [PyDataTables](../pydatatables/index.html), the [DataTables](https://datatables.net/) renderer. The two packages share their core functions (downsampling, formatting) through the `itables_core` package.

Other projects that bring AG Grid to Python include
[dash-ag-grid](https://dash.plotly.com/dash-ag-grid) (a Dash component
maintained by Plotly) and
[streamlit-aggrid](https://github.com/PablocFonseca/streamlit-aggrid)
(a Streamlit component).
